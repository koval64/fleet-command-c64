# Fleet Command C64

Retro-style browser naval tactics game.

## What's new (June 2026)

- Tryby gry: Klasyczny, Blitz, Taktyczny.
- Perki startowe: dodatkowy skan, salwa liniowa, mina, +1 salwa.
- Umiejętności taktyczne w HUD: Skan 3x3, Salwa liniowa, Mina, Podbij ryzyko.
- Cele rundowe z premiami do salw.
- Daily challenge z deterministyczną planszą dnia i lokalnym rekordem.
- Profil postępu zapisany lokalnie (gry/wygrane/rekordy daily).
- Ulepszony AI hard (agresywnie domyka trafione cele).
- Podpowiedzi ciepło/zimno po pudłach.
- Rozszerzone podsumowanie końca gry (celność, cele rundowe, daily score).
